### 数据库SQL语句

#### MYSQL5

```sql
SELECT
t1.`BIGINT_VALUE`,
t1.`CREATE_MAN`,
t1.`CREATE_TIME`,
t1.`DATETIME_VALUE`,
t1.`DECIMAL_VALUE`,
t1.`DOUBLE_VALUE`,
t1.`ID`,
t1.`INT_VALUE`,
t1.`NAME`,
t1.`OWNER_ID`,
t1.`OWNER_TYPE`,
t1.`STRING_VALUE`,
t1.`UPDATE_MAN`,
t1.`UPDATE_TIME`
FROM `EXTEND_STORAGE` t1 


```
